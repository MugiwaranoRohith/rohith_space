# Cosmic Exploration Website

A frontend-only space exploration website built with React, focused on clean UI and smooth interactions.

## Features
- Space-themed landing page with interactive sections
- Destination cards with modal details
- Smooth scrolling navigation and animated CTA
- Fully responsive layout

## Tech Stack
React · JavaScript · HTML · CSS

## Live Demo
https://your-live-link.vercel.app


## Getting Started

1. Clone the repository
   git clone https://github.com/your-username/cosmic-exploration.git

2. Navigate to the project folder
   cd cosmic-exploration

3. Install dependencies
   npm install

4. Start the development server
   npm start

The app will run on http://localhost:3000
